# 🌸 She Can Foundation - Frontend Internship Task

A modern and responsive single-page website created for the She Can Foundation Frontend Development Internship Task.

## 🚀 Live Demo

🔗 https://she-can-foundation-task-pi.vercel.app

---

## ✨ Features

- Responsive Design
- Modern Glassmorphism UI
- Dark / Light Mode Toggle
- Interactive Button
- Smooth Animations
- Floating Background Effects
- Clean & Well Commented Code
- Beginner Friendly Structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
